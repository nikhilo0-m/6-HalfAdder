📌 Project Overview

This project implements a Half Adder using Verilog HDL. A Half Adder is a basic combinational circuit that performs binary addition of two single-bit inputs and produces a Sum and a Carry output.
It is one of the fundamental building blocks in digital electronics and is widely used in arithmetic circuits like Full Adders .

⚙️ Features
1-bit binary addition
Two inputs: A, B
Two outputs: Sum, Carry
Implemented using Verilog HDL
Verified using testbench simulation

🚀 Tools Used
Verilog HDL
Xilinx Vivado / ModelSim
FPGA Simulation Tools

🎯 Applications
Digital arithmetic circuits
Building block for Full Adder
Computer architecture fundamentals

📖 Learning Outcome
Understanding combinational logic design
Implementation of basic arithmetic circuits
Verilog HDL coding practice
Testbench verification skills
